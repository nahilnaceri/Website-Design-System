Headings

Main Heading

h1's are very important you should only have one per page as it conveys the subject matter presented on a page

<h1> Style Below </h1> 
            
    h1 {
        font-weight: bold;
        font-size: 24px;
        font-family: monospace;
    }

Sub Headings

Subheadings are used to differentiate between different sections, but still catch the users attention

<h2> Style Below </h2> 
                
    h2 {
        font-weight: bold;
        font-size: 20px;
        font-family: monospace;
    }
<h3> Style Below </h3> 
                
    h3 {
        font-weight: bold;
        font-size: 18px;
        font-family: monospace;
    }
<h4> Style Below </h4> 
                
    h3 {
        font-weight: bold;
        font-size: 16px;
        font-family: monospace;
    }
